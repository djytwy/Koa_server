# Koa_server
主要是提供一个koa后端，vue前端的基础脚手架
为了减少包体积，部分资源使用了cdn加速去获取

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```

### 启动APP
```
npm run app
```
